Développement Web · 06

Site DRC

Site web complet conçu et développé pour le Dammarie Rugby Club dans le cadre d'une candidature en alternance — Chef de projet Web & Stratégie Digitale. Un projet from scratch pour démontrer mes compétences en conditions réelles, avec de vraies fonctionnalités pensées pour le club.

HTML / CSS JavaScript Supabase Authentification Espace admin Mini-jeu pronostics E-mail via site
Statut Projet personnel
Type Site web complet
Client cible Dammarie Rugby Club
Année 2026
01

Le contexte

Plutôt que d'envoyer un CV classique, j'ai choisi de développer un site web complet pour le Dammarie Rugby Club afin de candidater à une alternance en tant que Chef de projet Web & Stratégie Digitale. L'idée : prouver concrètement ce que je suis capable de faire, pas juste l'écrire sur du papier.

Le site a été pensé comme un vrai outil pour le club — pas une démo vide. Chaque fonctionnalité répond à un besoin réel : vendre des produits, engager les supporters, faciliter le contact, afficher les résultats.

« Un CV, ça s'oublie. Un site fonctionnel qu'on a codé pour vous, beaucoup moins. »

02

Les fonctionnalités

  • [ 01 ]

    Système de matchs & pronostics

    L'administrateur paramètre les matchs, les compositions et les scores via un espace dédié. Les supporters connectés peuvent pronostiquer les scores et s'affrontent dans un classement général. Backend géré via Supabase (base de données + authentification).

  • [ 02 ]

    Boutique officielle

    Redirection directe vers la boutique officielle du club, en partenariat avec Macron (équipementier officiel). Un bouton simple, bien intégré, pour ne jamais perdre un acheteur potentiel.

  • [ 03 ]

    Formulaire de contact

    Envoi de mail directement depuis le site, sans passer par un client mail externe. Le visiteur remplit un formulaire, le club reçoit le message. Simple et efficace.

  • [ 04 ]

    Localisation intégrée

    Carte interactive pour trouver le stade facilement — utile pour les nouveaux supporters, les adversaires et les partenaires du club.

  • [ 05 ]

    Authentification & espace admin

    Système de connexion complet via Supabase. L'administrateur accède à un back-office pour gérer les matchs, les scores et les compositions en temps réel.

  • [ 06 ]

    Pages éditoriales

    Histoire du club, actualités, présentation des équipes — un socle éditorial complet pour donner de la profondeur au site et fidéliser les visiteurs.

03

Stack technique

Le site est développé en HTML / CSS / JavaScript vanilla — pas de framework, tout from scratch, pour garder un contrôle total sur les performances et l'architecture.

La partie base de données et authentification repose sur Supabase : gestion des comptes utilisateurs, stockage des pronostics, des scores et des compositions de matchs en temps réel. Le formulaire de contact utilise un service d'envoi d'email côté serveur pour ne pas exposer les identifiants.

La carte de localisation est intégrée via une iframe Google Maps directement dans la page contact, sans dépendance supplémentaire.
